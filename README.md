Веб приложение для базы данных транспортной компании. Приложение включает в себя следующие функции:
1) Аутентификация пользователя с помощью логина и пароля с использованием PHP-сессии;
2) Соответствие пользовательского интерфейса приложения "бизнес-логике" предметной области;
3) Просмотр данных из основных таблиц БД;
4) Форма для добавление записей как минимум в одну таблицу БД;
5) Удаление записей как минимум из одной таблицы БД с контролем целостности (предотвращение удаления записи, если имеются связанные с ней записи);
6) Просмотр изображений, связанных с одной из таблиц БД;
